Access the course here: https://www.kaggle.com/learn/intro-to-programming

**Courses**

*   **Data types**: explore integers, floats, booleans, and strings
*   **Conditions and conditional statements**: modify how functions run, depending on the input
*   **Intro to lists**: organize data to work with it efficiently
    
Each notebook contains exercises that reflect my step-by-step learning progress.
